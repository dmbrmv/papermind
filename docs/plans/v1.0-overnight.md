## Batch A: Critical Blockers (3 tasks)

### A1. Fix transformers dependency for [ocr] extra
**File**: `pyproject.toml`
**Problem**: `transformers>=5.0` isn't on PyPI (pre-release). `pip install hydrofound[ocr]` installs 4.x which breaks.
**Fix**: Change to `transformers>=4.48` (works with `trust_remote_code=True`) and add a runtime version check in `glm_ocr.py` that gives a clear error if the installed version can't load GLM-OCR.
```python
# In glm_ocr.py _ensure_model(), after imports:
import transformers
if not hasattr(transformers, "GlmOcrConfig"):
    # Model needs transformers dev branch
    try:
        processor = AutoProcessor.from_pretrained(model_name, trust_remote_code=True)
    except Exception as exc:
        raise ImportError(
            f"GLM-OCR model requires transformers >= 5.0 (dev branch). "
            f"Installed: {transformers.__version__}. "
            f"Install with: pip install 'transformers @ git+https://github.com/huggingface/transformers.git'"
        ) from exc
```

### A2. Fix qmd_search cwd
**File**: `src/hydrofound/query/qmd.py`
**Problem**: `subprocess.run(cmd, ...)` has no `cwd`, searches wrong collection.
**Fix**: Add `cwd=str(kb_path)` to the subprocess call. One line.

### A3. Guard duplicate codebase ingestion
**File**: `src/hydrofound/cli/ingest.py`
**Problem**: Re-ingesting same codebase creates duplicate catalog entries.
**Fix**: Before `catalog.add(entry)`, check if entry ID exists and remove old one first (same pattern as package ingestion).

**Commit message**: `fix: 3 critical blockers — transformers dep, qmd cwd, codebase dedup`

## Batch D: Missing Features (5 tasks)

**Fix**: Add `abstract_inverted_index` to the `select` fields, reconstruct abstract text from the inverted index (OpenAlex stores abstracts as `{word: [positions]}` dicts).

### D3. catalog show --json flag
**File**: `src/hydrofound/cli/catalog.py`

**Fix**: Add `--json` flag to `show` that prints `catalog.json` contents.

### D4. BibTeX export
**File**: `src/hydrofound/cli/catalog.py` (new `export` command)
**Problem**: No citation export for reference managers.
**Fix**: New `hydrofound --kb <path> export bibtex` command that reads catalog entries with DOIs and outputs `.bib` format.

### D5. Paper dedup by title similarity
**File**: `src/hydrofound/ingestion/paper.py`

**Fix**: Before ingestion, check if any existing entry has >90% title similarity (using `difflib.SequenceMatcher`). If so, warn and skip (same pattern as DOI dedup).
